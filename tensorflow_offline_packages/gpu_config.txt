Configuration TensorFlow pour GPU A100 sur SUSE 15 SP7

CUDA Version: 11.8
cuDNN Version: 8.6

Prérequis système:
1. Driver NVIDIA installé (version >= 525.60.13 pour A100)
2. CUDA Toolkit 11.8 installé
3. cuDNN 8.6 installé

Vérification de la GPU:
- nvidia-smi (doit afficher l'A100)
- nvcc --version (doit afficher CUDA 11.8)

Variables d'environnement recommandées:
export LD_LIBRARY_PATH=/usr/local/cuda-11.8/lib64:$LD_LIBRARY_PATH
export CUDA_HOME=/usr/local/cuda-11.8
export PATH=/usr/local/cuda-11.8/bin:$PATH

Test TensorFlow GPU:
python3 -c "import tensorflow as tf; print('GPU disponible:', tf.config.list_physical_devices('GPU'))"
